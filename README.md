# Nanoparticle Diffusion in Polymer Networks

This repository contains the simulation analysis code for a study of
nanoparticle (NP) transport through polymer networks modelled by Langevin
dynamics. The system explores how steric confinement and electrostatic
interactions combine to produce three distinct diffusion regimes: free
Brownian diffusion at low polymer density, anomalous subdiffusion when the
mesh is dense enough to trap the particle, and hopping transport driven by
electrostatic binding at intermediate density. The analysis draws on mean
squared displacement, Van Hove correlation functions, velocity
autocorrelation, and machine learning classification, with direct relevance
to X-ray photon correlation spectroscopy (XPCS) experiments on cellulose
nanocrystal and nanofiber suspensions in polymer matrices.

## Physical model

The nanoparticle obeys the Langevin equation of motion:

$$m\ddot{\mathbf{r}} = -\gamma \dot{\mathbf{r}} + \mathbf{F}_\text{polymer}
 + \mathbf{F}_\text{elec} + \boldsymbol{\xi}(t)$$

where $\gamma$ is the friction coefficient, $\mathbf{F}_\text{polymer}$
encodes steric and viscoelastic interactions with the mesh,
$\mathbf{F}_\text{elec}$ is the screened Coulomb interaction, and
$\boldsymbol{\xi}(t)$ is Gaussian white noise.

| Parameter | Symbol | Values studied |
|---|---|---|
| Polymer volume fraction | phi | 0.00, 0.05, 0.15, 0.30, 0.50 |
| NP charge | Z (e) | 0.0, 1.0, 3.0, 6.0 |
| Independent runs per condition | | 8 |
| Total trajectories | | 160 |
| Frames per trajectory | | 15,000 |
| Saved time step | dt | 0.02 LJ time units |
| Box size | L | 50 sigma |

## Key results

- The anomalous diffusion exponent alpha drops from 0.97 at phi = 0.00 to
  below 0.65 at phi = 0.50 for all charge values, with electrostatic
  interactions suppressing alpha by an additional 0.10 to 0.15 at
  intermediate concentrations (see `figures/02_alpha_heatmap.png`).

- Hopping transport, characterised by elevated non-Gaussian parameter
  alpha_2 and frequent cage escapes, is confined to phi = 0.15 to 0.30
  with Z = 3 to 6: the particle is mobile enough to escape cages but
  electrostatically attracted to polymer strands
  (see `figures/05_phase_diagram.png`).

- Random Forest and Gradient Boosting classifiers both achieve greater than
  90% test accuracy in identifying diffusion regimes from trajectory features
  alone, with the anomalous exponent and non-Gaussian parameter as the most
  diagnostic observables (see `figures/06_confusion_matrices.png`).

## Diffusion regimes identified

| Regime | Description |
|---|---|
| free | MSD exponent alpha near 1, large D; dilute network, no significant trapping |
| subdiff | alpha well below 1; strong steric confinement by dense polymer mesh |
| hopping | Intermediate alpha but non-Gaussian displacements and frequent cage escapes; electrostatic binding at moderate density |
| mixed | Intermediate statistics; transitions between trapping and mobile states |

## How to reproduce

1. Clone the repository
2. `cd nanoparticle-diffusion`
3. `pip install -r requirements.txt`
4. `cd .. && python generate_trajectories.py`   (generates `data/`, approximately 10 min)
5. `cd nanoparticle-diffusion && jupyter notebook`
6. Run notebooks 01 through 06 in order.
